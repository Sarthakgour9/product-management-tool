About This Product Management System

A full-stack web application for efficient product management

📋

This Product Management System is a full-stack web application designed to manage product information efficiently. It allows users to perform essential CRUD (Create, Read, Update, Delete) operations on products through a clean and interactive interface.

🎯

Purpose of the Tool

The goal of this project is to demonstrate how modern web applications are built using Angular for the frontend and Node.js with PostgreSQL for the backend. It showcases how frontend applications communicate with APIs and manage dynamic data in real time.

⚡

Key Features

  • ✅ Add new products with details like name, price, category, and status
  • ✅ View all products in a structured table
  • ✅ Delete products from the database
  • ✅ Category and status selection using dropdown binding
  • ✅ Loading indicators and error handling
  • ✅ Responsive UI for better user experience
🛠️

Technologies Used

  • Angular Frontend framework for building dynamic user interfaces
  • Node.js Backend server and REST API development
  • TypeScript Strongly typed JavaScript for scalable applications
  • PostgreSQL Cloud-based relational database (NeonDB)
  • Express Fast and minimalist web framework for Node.js
🔄

The Angular frontend sends HTTP requests to the backend API built with Node.js and Express. The backend processes these requests and interacts with the PostgreSQL database to retrieve or modify product data. The results are then returned to the frontend and displayed dynamically without refreshing the page.

🚀

Project Objective

This project was created to understand full-stack development concepts such as Angular components, services, routing, API integration, and database connectivity. It also demonstrates how scalable enterprise applications are structured in real-world development environments.
